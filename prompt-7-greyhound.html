PART 2Copy · Paste · Run

The Master Command

Hit Copy command, paste it into Gemini exactly as is, then edit only the first line (set X = number of races; set scratches and track condition).

prompt_7_greyhound — master command
Prepare race analysis for races 1 through X. No scratches and track is fast. CRITICAL EXECUTION ORDER — PROMPT 7 (GREYHOUND):
  1. ENFORCE USER INPUT GATE: pause and collect the Pre-Race User Gate (Late Scratches, Reserves, Track Condition). STOP if these are missing — do not build the Speed Map or report until the user provides them or says “None.” If a reserve is promoted, recalculate Map Trouble Risk (MTR) for the neighboring boxes.
  2. EXTRACT GLOBAL STATS FIRST (CALIBRATION LOCK): parse the uploaded Track Statistics file before evaluating any dogs.
    • Lock Average Win Times: from the Grade/Distance table (e.g. 550m Grade A = 30.44s) set the “Field Average Best Time” (Par).
    • Lock Box Bias: from Post Position Stats, find the highest-win boxes at today’s distance — power the Box Fit Score (BFS).
    • Lock Kennel Heat: compare Kennel Records (Last 80) vs Overall to score Trainer/Kennel Heat (TKH).
  3. FORCE MODULE 1 GHOST PROTOCOL (FORENSIC SCRUB): scan last-race comments for trouble keywords (“Fell”, “KO”, “Crd”, “Bmp”, “Checked”) or beaten margins > 15 lengths. Apply PROTECTED STATUS: delete the troubled line, rate off the 2nd race back, and add the Hidden Merit bonus (+3 TPN-G).
  4. LOCK IN THE IMMUNITY HIERARCHY:
    • Elite Speed Guardrail: dogs with HLM or eBLI ≥ 8.5 cannot take catastrophic MTR penalties — cap their MTR at −0.5.
    • Rail-Hold Override: if Box 1 and Box 2 are both early pace, protect Box 1 (cap MTR −1.0) and penalize Box 2.
    • Failed-Fav Forgiveness: beaten favorites last start get a +2 TPN-G bonus before consistency is judged.
  5. ASSESS CHAOS & MAP DYNAMICS (MODULES 6 & 3): if the standard deviation of projected first splits is high (≥ 0.09s), activate the Chaos Adjuster — promote dogs with clear lanes (vacant box / outside flow) and suppress vulnerable favorites drawn in traffic.
  6. EXECUTE THE PB SAFETY NET (MODULE 5 RE-RANK): scan dogs ranked 5th–8th; if an outsider’s Personal Best is ≥ 0.15s faster than Par, FORCE PROMOTION to the Top 4 (Value / Saver slot). Don’t let recent poor form bury a fast dog.
  7. GENERATE BIMODAL REPORT: run the full TrackSmart AI Prompt 7 analysis — full breakdown for the Top 4 dogs ONLY; Ranks 5–8 as a Single-Line Scrub.
⚑ AI MUST NOT INCLUDE SOURCES IN REPORT OUTPUT.
Press Enter — Gemini locks the track stats, builds the Speed Map, and ranks your Top 4 dogs.

Tips for Best Results

Update for Scratches

For the most accurate analysis, always answer the Pre-Race Gate — late scratches, reserves and track condition.

Be Patient

Running a full card through the engine takes a few minutes. That’s normal — the AI is processing thousands of data points.

Managing Large Cards

Set X to the number of races you want. For very large cards, run a batch (e.g. 1 through 8), then start a fresh chat, re-upload, and set the next batch.

One Card per Chat

Always start a brand-new Gemini chat for a different track or day to avoid cross-contamination of track stats and box bias.
