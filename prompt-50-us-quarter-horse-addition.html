Pro Get The Most From It

Tips for Best Results

Update Scratches & Jockey Changes

Quarter Horse races are decided at the gate, so a late jockey change matters more here than in route racing. Always update the first line before you hit Enter.

Set the Track Condition Accurately

Fast, sloppy, or muddy changes Today's Par — and Par is what every Speed Index in the Gate Matrix is measured against.

Watch for the Chaos Survivor

A 💣 next to a #4 ranking means the Value Promotion Trigger fired — a live longshot with a clean break history and a Normalized TPN of 80+.

No Pace Figures — By Design

Prompt 50 deliberately omits E1/E2 pace fractions — QH sprints have no far turn to make them meaningful. If the AI invents one, that's a bug, not a feature.

Managing Large Cards

Set X to the number of races you want. For very large cards, run a batch (e.g. 1 through 8), then start a fresh chat, re-upload, and set the next batch.

One Card per Chat

Always start a brand-new Gemini chat for a different track or day to avoid cross-contamination of track biases.
